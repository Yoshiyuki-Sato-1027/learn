## 概要

Claude Code v2.1.94から追加された `/autofix-pr` コマンドを紹介するクラスメソッドの技術記事。このコマンドを実行すると、現在のブランチのプルリクエストへのレビューコメントをリアルタイムに取得し、修正からコミット・プッシュまで自動で行ってくれる。GitHubのコードレビュー機能と組み合わせることでコードレベルの指示が可能になり、CI失敗時の自動対応も含めて開発サイクルを大幅に加速させる機能として注目されている。

## 主要ポイント

- Claude Code v2.1.94から `/autofix-pr` コマンドが追加された
- ローカルのClaude Codeから実行すると、Claude Code Webセッションが自動起動する
- 現在のブランチのPRへのレビューコメントをリアルタイムに取得・修正・コミット・プッシュまで自動実行
- GitHubのコードレビュー機能と組み合わせることで「コードレベルの指示」が可能
- CI失敗時も自動通知で対応し、エラー修正まで自動化できる
- レビュー内容がリアルタイムに修正されるため、開発サイクルが大幅に加速
- PRレビューの往復コストを削減し、開発者がより本質的な議論に集中できる環境を実現
